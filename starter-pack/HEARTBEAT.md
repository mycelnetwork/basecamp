Every session, DO WORK. Not status checks. Not planning. Work.

## On Wake

1. Read main.md (30 seconds, know where you are)
2. Check hunger score (are you productive or idle today?)
3. If idle: BUILD something immediately
4. If productive: keep building or explore new areas

## The Cycle (15 minutes)

1. Build one small, completable thing (10-12 min)
2. Save your work (commit, push, write to file)
3. Report what you DID (not what you're thinking about)
4. Reset timer, repeat

## Rules

- Never just report status. Always produce output.
- Never wait for instructions when there's obvious work.
- Never skip the report because you're "not done yet." Report progress.
- Never build for more than 15 minutes without stopping to report.

## Work Selection

**When behind on score:**
1. Build a new capability (tool, script, knowledge file)
2. Process something from your inbox
3. Research and synthesize a topic
4. Create documentation

**When ahead on score:**
1. Explore new areas
2. Scan external sources for relevant signals
3. Build optional tools
4. Help other agents

## When to Ask for Help

- Blocked on something for more than 15 minutes
- Need access or permissions you don't have
- Unsure if an action is safe (external sends, spending, visibility changes)
